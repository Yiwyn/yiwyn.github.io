极限问题常用解决方法

2020-04-11

极限存在的条件

函数极限存在的充要条件:左右极限存在且相等

常用的解决方法

  • 洛必达法则

    当要求计算的函数满足 0/0 or ∞/无穷 时,可以对其上下求导.

  • 熟练使用等价无穷小

    在进行下一步操作时可以先看一下等价无穷小是否可以减少问题的复杂度

  • 直接带入

    简单粗暴,能直接带入计算的就带入.

  • 两个重要极限
    当x -> 0时 sinx/x=1 (1+x)^(1/x) = e

小技巧

  • 当x -> +∞ 时

    x+1 =x 显而易见 可以扩展到 x + C = x (无穷大的数不受影响)
    同理(x^2 + x )/(x+1) -> 则可以变成 (x^2 + x )/x ->进一步化简发现 (x+1)/1->x
    根据上面的式子我们可以推断出:
    当无穷和常数加减运算时可以直接省去常数. 同样的 而分子分母同时出现x的情况下 同时除x也可以得到 x+常数的情况 .

举个例子:(sqrt()表示根号)
lim(x ->+∞) x/[sqrt(x^2 +1) + x]
= x/[sqrt(x^2)+x] = x/(x+x) = 1/2
