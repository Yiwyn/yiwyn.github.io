SQL语句及其通配符

2020-05-26

SELET字段列表,[INTO 目标表]
FROM 名
WHERE 条件
[ORDER BY 字段]
[GROUP BY 字段 HAVING 条件]
如果字段列表用*表示,则表示所有字段
如果字段列表是从所有字段中选择部分字段,则是投影运算

更新:UPDATE 表名    set 字段     = 表达式[WHERE 条件]
    Eg1.UPDATE 考生表 set 录取 = "是" WHERE 分数>=100

插入:INSERT INTO 表名(字段列表) VALUES(内容列表)

删除:DELETE FROM 表名[WHERE 条件]

通配符:
        % :任意多字符,用于模糊查询,line,
        _:任意一个字符
